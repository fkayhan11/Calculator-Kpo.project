# Lab 7 – Configuration Files and Application Settings

Bu laboratuvar çalışmasında mevcut Calculator projesi geliştirilmiş ve uygulamaya yapılandırma (configuration) sistemi eklenmiştir.

Projede:
- config.json dosyası oluşturuldu
- ConfigManager sınıfı geliştirildi
- JSON tabanlı ayar sistemi eklendi
- Arayüz ayarları dinamik hale getirildi
- Dark mode ayarları config üzerinden yönetildi
- Font ve pencere boyutları config dosyasından okunmaya başlandı
- Cursor ayarları config üzerinden kontrol edildi
- Config error handling sistemi geliştirildi
- Otomatik config oluşturma sistemi eklendi
- MVC mimarisi korunarak proje genişletildi

Kullanılan teknolojiler:
- Python
- Tkinter
- JSON
- OOP
- MVC Architecture
- Design Patterns
- Git / GitHub

Kullanılan tasarım desenleri:
- MVC
- Factory Method
- Decorator
- Strategy

Yeni eklenen dosyalar:
- config.json
- config_manager.py

Lab 7 sonunda proje daha esnek, yapılandırılabilir ve profesyonel hale getirilmiştir.
