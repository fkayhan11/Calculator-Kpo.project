Calculi is a minimalist, precision-focused calculator application. What makes this project unique is its **iterative development process**. Instead of being built all at once, this application was developed step-by-step, utilizing different Git branches to isolate, test, and implement features progressively. 

This repository serves not just as a functional tool, but as a demonstration of structured version control and scalable software architecture.

## 🌿 Development Evolution

The project history is divided into specific branches, reflecting its step-by-step growth:

* **`phase-1-core`**: Implementation of basic arithmetic operations (addition, subtraction, multiplication, division).
* **`phase-2-advanced`**: Integration of advanced mathematical functions (percentages, square roots, exponentiation).
* **`phase-3-ui`**: Design and implementation of the minimalist user interface.
* **`main`**: The final, stable fusion of all progressive branches.

## ✨ Key Features

* **Clean Interface:** A distraction-free, minimalist design focusing on usability.
* **Progressive Architecture:** Codebase structured to easily accept new mathematical modules.
* **Error Handling:** Robust protection against common mathematical errors (e.g., division by zero).
* **History Tracking:** Capable of maintaining a brief history of recent calculations.
