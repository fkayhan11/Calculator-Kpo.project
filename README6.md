# Lab 6 – Calculator GUI Improvement

Bu laboratuvar çalışmasında mevcut Python tabanlı Calculator projesi geliştirilmiştir.

Projede:
- Tkinter ile modern GUI oluşturuldu
- Menü sistemi eklendi
- About penceresi oluşturuldu
- Dark Mode desteği eklendi
- Cursor özelleştirildi
- Klavye desteği geliştirildi
- Resources klasörü oluşturuldu
- MVC mimarisi korundu

Kullanılan tasarım desenleri:
- MVC
- Factory Method
- Decorator
- Strategy

Kullanılan teknolojiler:
- Python
- Tkinter
- Pillow
- Git / GitHub
